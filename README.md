# 中文-西班牙语双语对照国际贸易术语库（静态站点）

这是一个可直接部署的静态网页术语库示例：数据在 `data/terms.json`，页面提供双语对照、检索、分类筛选、按业务流程浏览与详情弹窗。

当前内容已扩展为偏实务的版本，重点覆盖：

## 分类栏目

页面分类下拉目前统一为三个一级类目：

- **业务流程**：包含外贸、物流、关务、结算金融、合同、保险、拉美清关、拉美贸易协定、电商与商务文化、IMPARGO 运输与物流词汇等业务流程相关术语，共 466 条。
- **企业组织**：包含企业成立、注册、公司治理、职能部门、职位名称等企业组织相关术语，共 51 条。
- **摩配汽配**：包含摩托车配件和汽车配件中西语品名，共 221 条。

## 自定义新增术语

页面已支持自定义新增术语：

- 点击页面中的「新增术语」按钮，可录入分类、缩写、中文术语、西语术语、标签、双语释义和例句。
- 新增术语会保存在当前浏览器本地存储中，刷新页面后仍可检索和查看。
- 自定义术语会参与分类筛选、关键词搜索和详情弹窗展示。
- 点击「导出自定义术语」可下载本地新增词的 JSON 文件，便于备份或后续合并进 `data/terms.json`。
- 由于这是纯静态站点，浏览器端新增词不会自动写回服务器上的 `data/terms.json`。

## 本地运行

由于浏览器的 `fetch()` 限制，需要用本地 HTTP 服务打开（不要直接双击 `index.html`）。

1. 进入项目目录
2. 启动一个简单静态服务器（任选其一）

```bash
python3 -m http.server 8000
```

然后在浏览器访问：

`http://localhost:8000/`

## 数据结构（terms.json）

`data/terms.json` 是一个数组，每条术语建议包含：

- `id`：唯一标识（推荐用英文与下划线，便于稳定引用）
- `abbr`：缩写（可为空）
- `category`：分类（用于筛选）
- `tags`：标签数组（用于快速定位与检索）。建议使用 `中文/ES` 形式，既便于中文查看，也便于西语检索（例如：`"海运/Marítimo"`）。
- `source`：来源说明（建议标注"内部整理/客户确认/法规链接"等）
- `updatedAt`：更新时间（字符串即可）
- `zh`：中文字段
  - `term`：中文术语
  - `aliases`：中文别名数组（可选）
  - `definition`：中文解释
  - `example`：中文例句（可选）
  - `commonMistake`：常见误区（可选）
  - `practicalTip`：实务提醒（可选）
  - `spainExample`：场景例句（可选，拉美术语中为拉美场景例句）
  - `note`：中文备注（可选）
- `es`：西班牙语字段（结构同 `zh`，建议使用自然西语表达）
  - `commonMistake` / `practicalTip` / `spainExample`：对应的西语版扩展字段

你可以直接复制现有条目，新增更多术语即可。
